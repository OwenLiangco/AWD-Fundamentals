Variables & Input and Output

Variables are used to store data or values, like numbers, text, or other information. They let you reuse and manipulate this data in the program.

While Input and Output Statements allows users to input and provide data to the program, while output displays results or feedback. Together, they make programs interactive, Go ahead and input something!

Conditionals

Conditionals let your program make decisions based on specific conditions. For example, try putting a number and it will determine if its an odd number or an even number.

Looping

Loops allow your program to repeat actions or process data multiple times efficiently, click the button it will create a list of numbers from 1-5! (In looping it will become false when it hits a certain statement, and it will execute until that statement has been met.)
